		注意： 静态变量成员不能序列化，它不属于对象，只属于类

	  ---ObjectInputStream  ：

	     // 创建反序列化字节输入流
		FileInputStream fis = new FileInputStream("E://infomation");
		// 穿件反序列化流
		ObjectInputStream ois = new ObjectInputStream(fis);
		// 得到对象
		Object obj = ois.readObject();
		// 关闭流
		ois.close();
		// 输出对象
		System.out.println(obj);
